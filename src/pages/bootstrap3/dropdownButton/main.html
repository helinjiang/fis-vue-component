DropdownButton 按钮式下拉菜单

1. 使用

略

2. 选项配置

2.1 slot、dropdownMenu 和 menuItem

dropdownButton 下拉按钮组件包含两块块 slot，分别是：

  • （省略）：不设置 slot 的默认为按钮中的内容
  • dropdown-menu：下拉菜单区域

下拉菜单采用 dropdownMenu 组件，其中包含的每一项菜单为 menuItem 组件。menuItem 组件可选参数为：

  • header：Boolean 类型，非必须，是否为下拉菜单的标题
  • divider：Boolean 类型，非必须，是否为下拉菜单的分割线
  • disabled：Boolean 类型，非必须，是否为禁用的菜单项
  • href：String 类型，非必须，跳转链接的href
  • target：String 类型，非必须，跳转链接的target
下拉框 1. 超链接 GO QQ.COM 2. 普通行为 Action Disabled Action End link 

    <bs3-dropdown-button>
        <span>下拉框</span>
        <bs3-dropdown-menu slot="dropdown-menu">
            <bs3-menu-item header>1. 超链接</bs3-menu-item>
            <bs3-menu-item href="http://www.qq.com" target="_blank">GO QQ.COM</bs3-menu-item>
            <bs3-menu-item divider></bs3-menu-item>
            <bs3-menu-item header>2. 普通行为</bs3-menu-item>
            <bs3-menu-item @click="testAction">Action</bs3-menu-item>
            <bs3-menu-item disabled>Disabled Action</bs3-menu-item>
            <bs3-menu-item divider></bs3-menu-item>
            <bs3-menu-item>End link</bs3-menu-item>
        </bs3-dropdown-menu>
    </bs3-dropdown-button>

2.2 bs-style

（颜色）样式，String，非必须，可选值为：

  • default：默认值，默认样式
  • primary：首选项
  • success：成功
  • info：一般信息
  • warning：警告
  • danger：危险
下拉框{{curStyle}} Action 

    <bs3-dropdown-button bs-style="success">
        <span>下拉框</span>
        <bs3-dropdown-menu slot="dropdown-menu">
            <bs3-menu-item>Action</bs3-menu-item>
        </bs3-dropdown-menu>
    </bs3-dropdown-button>

2.3 size

按钮大小，String，非必须，可选值为：

  • （省略）：默认值，默认大小不用设置
  • lg：大按钮
  • sm：小按钮
  • xs：超小尺寸
下拉框{{curSize}} Action 

    <bs3-dropdown-button size="lg">
        <span>下拉框</span>
        <bs3-dropdown-menu slot="dropdown-menu">
            <bs3-menu-item>Action</bs3-menu-item>
        </bs3-dropdown-menu>
    </bs3-dropdown-button>

2.4 disabled

是否可点击，Boolean，默认为 false。

下拉框 Action 

    <bs3-dropdown-button disabled>
        <span>下拉框</span>
        <bs3-dropdown-menu slot="dropdown-menu">
            <bs3-menu-item>Action</bs3-menu-item>
        </bs3-dropdown-menu>
    </bs3-dropdown-button>

2.5 dropup 和 dropdown

向上弹出菜单还是向下弹出菜单，二选一，Boolean，默认为 dropdown: true。

下拉框 Action 

    <bs3-dropdown-button dropup>
        <span>下拉框</span>
        <bs3-dropdown-menu slot="dropdown-menu">
            <bs3-menu-item>Action</bs3-menu-item>
        </bs3-dropdown-menu>
    </bs3-dropdown-button>
