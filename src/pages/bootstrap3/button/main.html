Button按钮

1. 使用


    import Bs3Button from '/components/bootstrap3/button'

    export default {
        template: '<div><bs3-button bs-style="default">（默认样式）Default</bs3-button></div>',
        components: {
            Bs3Button
        }
    };

2. 选项配置

2.1 bs-style

（颜色）样式，默认值为 default

（默认样式）Default （首选项）Primary （成功）Success （一般信息）Info （警告）Warning （危险）Danger （链接）Link 

    <!-- Standard bs3-button -->
    <bs3-button bs-style="default">（默认样式）Default</bs3-button>

    <!-- Provides extra visual weight and identifies the primary action in a set of bs3-buttons -->
    <bs3-button bs-style="primary">（首选项）Primary</bs3-button>

    <!-- Indicates a successful or positive action -->
    <bs3-button bs-style="success">（成功）Success</bs3-button>

    <!-- Contextual bs3-button for informational alert messages -->
    <bs3-button bs-style="info">（一般信息）Info</bs3-button>

    <!-- Indicates caution should be taken with this action -->
    <bs3-button bs-style="warning">（警告）Warning</bs3-button>

    <!-- Indicates a dangerous or potentially negative action -->
    <bs3-button bs-style="danger">（危险）Danger</bs3-button>

    <!-- Deemphasize a bs3-button by making it look like a link while maintaining bs3-button behavior -->
    <bs3-button bs-style="link">（链接）Link</bs3-button>

2.2 bs-size

尺寸大小

（大按钮）Default bs3-button （默认尺寸）Default bs3-button （小按钮）Default bs3-button （超小尺寸）Default bs3-button 

    <bs3-button bs-size="lg">（大按钮）Default bs3-button</bs3-button>

    <bs3-button>（默认尺寸）Default bs3-button</bs3-button>

    <bs3-button bs-size="sm">（小按钮）Default bs3-button</bs3-button>

    <bs3-button bs-size="xs">（超小尺寸）Default bs3-button</bs3-button>
