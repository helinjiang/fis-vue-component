Button按钮

1. bs-style

（颜色）样式，默认值为 default


    <!-- Standard button -->
    <button bs-style="default">（默认样式）Default</button>

    <!-- Provides extra visual weight and identifies the primary action in a set of buttons -->
    <button bs-style="primary">（首选项）Primary</button>

    <!-- Indicates a successful or positive action -->
    <button bs-style="success">（成功）Success</button>

    <!-- Contextual button for informational alert messages -->
    <button bs-style="info">（一般信息）Info</button>

    <!-- Indicates caution should be taken with this action -->
    <button bs-style="warning">（警告）Warning</button>

    <!-- Indicates a dangerous or potentially negative action -->
    <button bs-style="danger">（危险）Danger</button>

    <!-- Deemphasize a button by making it look like a link while maintaining button behavior -->
    <button bs-style="link">（链接）Link</button>

2. bs-size

尺寸大小


    <button bs-size="lg">（大按钮）Default button</button>
    <button>（默认尺寸）Default button</button>
    <button bs-size="sm">（小按钮）Default button</button>
    <button bs-size="xs">（超小尺寸）Default button</button>
